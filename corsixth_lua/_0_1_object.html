function Object:getRenderAttachTile()

Get the primary tile which the object is attached to for rendering

For objects which attach to a single tile for rendering, this method will return the X and Y Lua world coordinates of that tile. For objects which split their rendering over multiple tiles, one of them is arbitrarily designated as the primary tile, and its coordinates are returned.
